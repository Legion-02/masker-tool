# Masker

Masker is a simple **hashing and verification tool** for pentesting and security projects.  
It supports multiple hashing algorithms and provides a clean banner for style.

## Usage
```bash
python3 masker.py hash sha256 "hello"
python3 masker.py verify sha256 "hello" 2cf24dba5fb0a30e26e83b2ac5b9e29e1b161e5c1fa7425e73043362938b9824
```
